### Шаги для развёртки приложения в Docker
1. Клонирование репозитория

```git clone https://github.com/OrneysOriginal/test-task.git```

2. Переход в репозиторий проекта

```cd test-task```

3. Запустить docker

```docker-compose up -d```

## Зависимость
python3.12

## Скачать результаты нагрузочного тестирования

```locust -f tests/locust_test.py --headless -u 100 -r 1 --csv=foo -t 30s```

--headless указывает на отсутствие веб интерфейса

-u указывает количество пользователей

-r скорость увелечения пользователей

-t время через которое тестирование завершиться и запишеться в файл

--csv название группы файлов после тестирования
